# **App Name**: FurniFind

## Core Features:

- Add Furniture: Allow users to add new furniture items with photos, prices, and other details.
- Edit Furniture: Enable users to edit the details of existing furniture items.
- View Furniture Details: Display furniture photos, prices, and other details in an appealing format.
- Contact Option: Provide a contact option for each furniture item to connect buyers and sellers.

## Style Guidelines:

- Primary color: Warm beige (#F5F5DC) to evoke a sense of comfort and home.
- Background color: Very light beige (#FAFAF0), a close variation on the primary to offer subtle variation.
- Accent color: Soft brown (#A0522D) to complement the primary color and provide contrast.
- Font pairing: 'Playfair' for headlines (serif) and 'PT Sans' for body (sans-serif) provide a balance of elegance and readability.
- Use simple, modern icons to represent furniture categories and functionalities.
- Clean, grid-based layout to showcase furniture images and details effectively.
- Subtle transitions when navigating between furniture items and details.
